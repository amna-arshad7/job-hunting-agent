# Reflection - Job-Hunting Agent

## What I Built
A file-driven Job-Hunting Agent that reads job posters, resumes,
and knowledge base files to help with job search preparation.

## Features Implemented
- File reading from input_jobs, input_resumes, input_kb folders
- Keyword extraction and skill matching
- Skill gap report generation
- Resume tailoring suggestions
- Interview question generation from KB
- Application tracker with CSV
- Reminder generation

## What I Learned
- How to build a basic AI agent using the GAME framework
- File handling in Python
- How to structure a real project with folders
- GitHub version control for project submission

## Challenges Faced
- Reading and combining multiple files
- Matching skills accurately from text
- Organizing output files properly

## Unique Feature Added
- Menu-based interface so user can select specific job
- Reminder urgency system showing overdue, today, this week

## Tools Used
- Python
- GitHub
- VS Code
